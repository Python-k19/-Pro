## 🌟 О программе

**Нейро-Озвучка Pro** — это профессиональный инструмент для синтеза речи, который превращает любой текст в естественно звучащее аудио. Программа использует передовые нейросетевые голоса Microsoft Azure, которые раньше были доступны только корпоративным клиентам за большие деньги.

### 💡 Зачем это нужно?

- **Контент-мейкерам** — озвучка постов, подкастов, видео
- **Студентам** — превращение конспектов в аудиолекции
- **Писателям** — проверка звучания диалогов на слух
- **Бизнесу** — голосовые приветствия и инструкции
- **Людям с нарушениями зрения** — доступ к текстовому контенту

## 🛠️ Технологии

| Компонент | Технология | Зачем |
|-----------|------------|-------|
| **Синтез речи** | Edge-TTS + Microsoft Azure | Нейросетевые голоса премиум-класса |
| **Обработка аудио** | Pydub + FFmpeg 8.1.1 | Фильтры, склейка, эффекты |
| **Воспроизведение** | Pygame | Встроенный плеер с паузой |
| **Интерфейс** | CustomTkinter | Современный тёмный дизайн |
| **Метаданные** | Mutagen | ID3-теги в MP3 файлах |
